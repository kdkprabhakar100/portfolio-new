# Prabhakar Modern Portfolio — Phase 1

This starter includes:

- Animated system loader
- Responsive futuristic navigation
- Cursor-reactive 3D globe
- GSAP hero entrance animation
- Grid, glow and grain visual system
- Responsive mobile layout
- Reduced-motion accessibility support
- A placeholder project section for the next phase

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Main files

- `src/app/page.tsx` — page assembly
- `src/components/SystemLoader.tsx` — opening loader
- `src/components/Hero.tsx` — hero content and GSAP animation
- `src/components/GlobeScene.tsx` — React Three Fiber scene
- `src/app/globals.css` — complete visual styling
